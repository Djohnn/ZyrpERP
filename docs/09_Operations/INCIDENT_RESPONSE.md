# Incident Response

| Severidade | Critério | Resposta inicial |
|---|---|---|
| SEV-1 | Vazamento cross-tenant, perda ampla ou indisponibilidade crítica | imediata; comando único e comunicação executiva |
| SEV-2 | Capability crítica degradada sem workaround aceitável | mobilização prioritária |
| SEV-3 | Impacto limitado com workaround | horário operacional |
| SEV-4 | Defeito menor ou observação | backlog priorizado |

Fluxo: detectar → classificar → conter → comunicar → recuperar → validar → postmortem sem culpados. Evidências são preservadas. Incidentes de privacidade seguem avaliação LGPD. Ações corretivas têm owner e prazo.
