# Frontend

Aplicação Web React de administração e operação online do Zyrp.

- Consome exclusivamente a API em `/api/v1` do backend Django/DRF.
- TLS obrigatório fora do ambiente local.
- Não armazena segredos no bundle; autenticação via sessão segura.

Status: Sprint 0 reserva o diretório. Implementação detalhada em sprints
posteriores conforme a PRD mestre. Este diretório mantém apenas responsabilidade
e convenções de alto nível até o desenvolvimento iniciar.
