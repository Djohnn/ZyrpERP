# PDV

Aplicativo Electron/React/SQLite do caixa desktop do Zyrp.

- Online por padrão, contingência offline restrita (ADR-003).
- Journal SQLite WAL persistente; sincronização idempotente com o backend.
- Dispositivos cadastrados e credenciais curtas revogáveis (SEC-001).

Status: Sprint 0 reserva o diretório. Implementação detalhada em sprints
posteriores. Este diretório mantém responsabilidade e convenções de alto nível
até o desenvolvimento iniciar.
